-->Στην Admin έχω φτιάξει το menu που θα βγάζει στον admin (μόνο τα println). Απο κάτω στα if έχω βάλει τις μεθόδους που πρέπει να φτιάξουμε για να γίνεται η κάθε λειτουργία.jj
